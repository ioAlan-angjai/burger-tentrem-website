# Burger Tentrem - Website Restoran

> Tugas Pemrograman Berbasis Platform | HTML, CSS, JavaScript

![Burger Tentrem Screenshot](image.png)

## Tentang Proyek

**Burger Tentrem** adalah website restoran burger yang menyajikan informasi tentang menu, layanan, dan lokasi. Website ini dibangun sebagai tugas pemrograman berbasis platform dengan tampilan yang responsif dan modern.

### Fitur Utama
- Navigasi menu interaktif (HOME, MENU, LAYANAN, LOCATIONS, CONTACT)
- Tampilan story brand "The Story of Burger Tentrem"
- Informasi tagline: *"Panggang Dengan Cinta, Disajikan Dengan Bangga Sejak 1987"*
- Cuaca real-time display (89°F | Sunny)
- Search functionality
- Desain responsif untuk berbagai perangkat

## Teknologi yang Digunakan

| Teknologi | Fungsi |
|-----------|--------|
| **HTML5** | Struktur halaman website |
| **CSS3** | Styling, layout, dan responsivitas |
| **JavaScript** | Interaktivitas (search, dynamic weather, dll) |
